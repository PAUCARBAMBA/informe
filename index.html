Informe Técnico de Levantamiento Topografico

Realizado con DJI Mavic 3 Pro Enterprise y Sistema PPK

1. Introducción

El presente informe detalla el levantamiento fotogramétrico realizado mediante tecnología drone y sistema PPK, para la generación de productos cartográficos de alta precisión. El proyecto se realizó siguiendo estándares internacionales de calidad y precisión en la captura y procesamiento de datos geoespaciales.

2. Objetivos

2.1 Objetivo General

Realizar el levantamiento fotogramétrico del área de estudio mediante tecnología drone y sistema PPK para la generación de productos cartográficos de alta precisión.


2.2 Objetivos Específicos

  • Generar un ortomosaico georreferenciado del área de estudio
  • Curvas de Nivel cada 0.5 metros
  • Crear un modelo digital de superficie de alta precisión
  • Crea un modelo Digital de Terreno de alta precision

5. Metodología

El levantamiento se realizó mediante la metodología PPK (Post-Processed Kinematic), siguiendo las siguientes etapas:


  1. Planificación del vuelo y establecimiento de puntos de control de validacion
  2. Captura de datos con drone y receptores GNSS
  3. Post-procesamiento de datos GNSS
  4. Procesamiento fotogramétrico
  5. Control de calidad y validación de resultados

6. Equipos Utilizados

  • Drone: DJI Mavic 3 Pro Enterprise
  • Receptores GNSS: 2 unidades Emlid RS3 y 2 unidades South G7
  • Software de procesamiento Fotogrametrico: Pix4D Enterprise Version 4.5.6
  • Software de procesamiento Geodesico: Trimble Business Center
